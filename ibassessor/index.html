Why it helps

Built around IB marking criteria for precise, efficient feedback

Dynamic Annotation Tools

Highlights, underlines, strikethroughs, boxes, and freehand drawing, all referencing IB standards. Mark key evidence, errors, or points of concern directly on the student's work, paired with text comments so your marking rationale is always clear.

Highlights, Wavy Underlines & Text Comments

Annotate through an IB assessment lens: highlight strong expression, flag logical gaps with wavy underlines, and anchor specific feedback with on-screen comments. Every mark tied to IB scoring logic.

Efficient Assessment Material Management

Quickly locate specific text using the Find tool, and bookmark important pages or sections for easy navigation. Manage your marking flow efficiently without losing your place, making long documents easier to handle.

Summarising Feedback

It is possible to give off-page comments - summative comments - that are linked to the current page of the document. Additionally, you can enter marks manually for each assessment criterion and add a summary comment that reflects the overall judgement, providing a complete feedback-giving experience.

Annotation showcase

Not just icons, an annotation library that mirrors real IB assessment practice

Every annotation in IbAssessor is built around IB official marking scenarios and criteria: evaluation, evidence, expression, calculation, structure, definition, diagrams, and error carried forward. Teachers can express judgements that map directly to IB scoring dimensions, and students can clearly understand what each mark means and how to improve, making feedback faster and more meaningful.

Install

Installation Guide

Click "Download for macOS" and follow the steps below. macOS may show a security prompt the first time you open a locally distributed app. This is normal.

1

Download the installer

Click the "Download for macOS" button to save the .dmg installer.

2

Drag to Applications

Open the downloaded .dmg file and drag IB Marking Tool into your Applications folder.

3

First launch

If macOS says it cannot verify the developer, go to System Settings → Privacy & Security and allow the app to open, or Control‑click the app and choose "Open".

4

Start marking

Import a PDF, choose your tools and annotations aligned with IB criteria, and export the marked PDF when you're done.

How to use

User Guide

Import a file

Open IbAssessor and import the student essay. PDFs can be annotated directly

Choose an annotation tool

The left toolbar offers highlights, wavy underlines, and arrow comment boxes. Select a tool and place annotations based on IB assessment requirements.

Use IB Annotations

Pick from the annotation library: Criterion badges, Good Point, Reference Needed, and more, all matched to IB marking standards. Click anywhere on the page to place an annotation. Drag, reposition, or delete as needed.

Edit feedback

Double-click or select a text comment box to edit its content. Drag any annotation to a more accurate position to refine its meaning.

Find and bookmark

Use the Find tool to search for keywords across the document, and add bookmarks to jump directly to key sections. This keeps your marking organised and efficient.

Enter marks and summary

For each IB criterion, manually enter the earned score and maximum score, and write an overall summary comment. This final mark and feedback summary can be exported together with the annotated PDF.

Running out of page space

When comments reach the edge of the page, the tool automatically expands the visible margin to give longer IB aligned feedback room to breathe, without disrupting your annotations.

Export to PDF

Once you're satisfied with your IB aligned annotations, export to PDF and send the marked up essay to the student or save it to your teaching records.
